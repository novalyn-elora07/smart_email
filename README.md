## Overview
This project focuses on building an AI-based system to classify enterprise
customer support emails and assign urgency levels.

Milestone 1 is dedicated to preparing a labeled and cleaned email dataset
for training NLP models.

## Work Completed
- Generated enterprise-style email dataset (~5200 emails)
- Categorized emails into complaint, request, feedback, spam
- Assigned urgency levels (high, medium, low)
- Removed noise such as greetings, signatures, and special characters
- Normalized and cleaned text data
- Produced model-ready dataset for NLP training

## Dataset Features
- email_id
- email_text
- category
- urgency
- timestamp

## Tools Used
Python, Pandas, NumPy, VS Code
